Jenkins 2.0 is an upcoming major new version of Jenkins.

Highlights

Pipeline as Code keeps your configuration DRY
New pipeline functionality in Jenkins allows you to define your job configuration in a textual format, version controlled along side your source code. It lets you define delivery pipelines with ease, copes well when pipelines get complex, and far less repetitive when you have lots of repositories.
Out of the box experience: You are in good hands
Jenkins 2.0 starts off with much more modern and richer set of functionalities that are catered to today's development best practices. These functionalities are carefully chosen from the vast pool of available Jenkins plugins by Jenkins developers and tested together for better quality.
Improved user interfaces
A number of small but useful improvements are made to the user interfaces that you touch everyday, such as the freestyle job configuration pages.
All the Jenkins goodness you've accustomed & loves
Jenkins 2.0 is a drop-in replacement of Jenkins 1.x and fully backward compatible. All the plugins and existing configuration should run as is.
